How to Play

You are dealt 4 cards from a standard deck. Your goal is to combine all four card values using +, -, *, / to make an expression that equals 24.

  • Drag cards to rearrange their order
  • Click the operator circles between cards to cycle through +, -, *, /
  • Use the < > arrows to change how the expression is grouped (parentheses)
  • The colored brackets below the cards show evaluation order — 1st, 2nd, 3rd

Scoring:

  • Submit a correct answer: +1 point
  • Skip when a solution exists: -1 point
  • Skip when no solution exists: no penalty
  • Hint reveals a solution: -1 point

Card values: A=1, 2-10 face value, J/Q/K=10. The game ends when the deck runs out.
